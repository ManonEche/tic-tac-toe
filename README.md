Plongez dans un univers de grilles uniques ! Chaque partie est un nouveau défi, avec des plateaux et pions aux formes variées. Pour les esprits stratèges qui aiment repousser leurs limites.


## Technologies

**Client:** HTML, CSS, TailwindCSS, NextJS


## Usage

- 2 joueurs
- Jouer en local
- Placer des pions
- Gagner la partie
- Changer de plateau
